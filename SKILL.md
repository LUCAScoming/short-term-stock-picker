# 短线选股策略 (AKShare版)

## 策略条件

| # | 条件 | 说明 |
|---|------|------|
| 1 | 流通市值 | ≤ 150亿 |
| 2 | 涨停 | 近20个交易日有涨停 |
| 3 | ST | 排除ST/*ST股票 |
| 4 | 板块 | 大A(主板)、深证、创业板、科创板 |
| 5 | 主力净流入 | 近3日资金估算 |
| 6 | 换手率 | ≤ 10% |

## 使用方式

### 运行选股脚本

```bash
~/.venv/tushare/bin/python3 ~/.openclaw/workspace/skills/short-term-stock-picker/scripts/pick_stocks.py
```

### 查看结果

结果保存在 `result.csv`，包含：
- 代码、名称
- 涨停次数(近20日)
- 流通市值、总市值
- 换手率
- 封板资金
- 所属行业、板块

## 数据来源

- 使用 **AKShare** 接口，无需API Token
- 涨停数据来源：东方财富涨停股池
- 市值数据来源：东方财富

## 注意事项

- AKShare数据来自东方财富，部分接口可能有网络限制
- 主力净流入为估算值，仅供参考
- 建议在交易日下午4点后运行，数据更完整
- 封板资金越大，主力封板意愿越强
